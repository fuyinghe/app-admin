前端模型函数文档

一、使用方法

1，在指定的容器中生成模型数据

$().modelManager([Object])

2，销毁

$().modelManager('destroy')

3，配置checkbox为true时，获得选择行，返回一个jsonlist,list中每个json返回当前行所有数据

$().modelManager('getSelections')

4，单独使用模块（提供add/edit/view）

$().modelManager('add/edit/view',{
    method:"post",
    modelApi:"",
    saveApi:"",
    viewApi:"",
    saveBtn:null,//将为页面上的该元素绑定保存事件
    callback:function(data){
        //成功弹出框点击确定之后的回调事件
    }
})
